# WorkBench 2.0.0 Release Notes

WorkBench 2.0.0 closes the full roadmap with eight completed phases.

## Highlights

- Unified provider support for Ollama, OpenAI, Anthropic, and Gemini.
- CLI and dashboard controls for provider switching, health checks, and model selection.
- Secret-backed API key storage outside the repository.
- Ollama remains the default local fallback provider.

## Roadmap completion

- Phase 1: Conversation history per task
- Phase 2: Agentic auto mode with task decomposition
- Phase 3: CI loop and test runners
- Phase 4: Web dashboard
- Phase 5: Workspace / multi-project mode
- Phase 6: Analytics and usage statistics
- Phase 7: Event hooks / Telegram notifications
- Phase 8: Multi-provider architecture
